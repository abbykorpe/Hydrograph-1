Limit Validations

Accelero Release Version 1.0

Validations for the Limit component are in place for each property present on the Properties window.

General Properties:

  • Primary Keys Fields -Key fields cannot be empty

    Also, Users are allowed to type only alphanumeric field names.

  • Secondary Key Fields -Secondary Key fields cannot be empty

    Also, Users are allowed to type only alphanumeric field names.

  • Phase -Phase has a default value of 0. However, blank text box results in error validation and is not allowed.
