Transform

Hydrograph Release Version 1.0

Transform component allows the user to apply custom transformations on the input one at a time. It is is present under the Transform category in the component palette. It contains only one input and one output port.

User is provided with options to create Java class that specify the Transform operation which needs to be performed. Optionally, user can also use the pre-defined standard Operation Class provided in the Operation Class section of the Help content.

The Transform component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
