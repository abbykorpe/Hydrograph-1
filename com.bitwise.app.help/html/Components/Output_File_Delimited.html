Output File Delimited

Accelero Release Version 1.0

Output File Delimited as the name suggests writes a delimited file as output and is present under the Output category in the component palette

This component is used to write a delimited text file. It accepts the delimiter from the user in the component properties and has a single input port.

The Output File Delimited component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
