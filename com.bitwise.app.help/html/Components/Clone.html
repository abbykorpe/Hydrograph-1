Clone

Accelero Release Version 1.0

Clone creates multiple copies of a single input on the output. It is present under the Straight Pull category in the component palette since no transformations are performed on the incoming data.

Clone,has a single input port and can transmit copy of the input data on one or more output ports. It does not have an unused port on it.

The Clone component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
