Pre-requisites for Windows

Hydrograph Release Version 1.0

Setup Requirements on developer machine: Operating system - Windows 7/8 64 bit

  • JDK 1.7 64 bit

  • RAM: Hydrograph tool requires minimum 1 GB of RAM hence developer machine should have minimum 4 GB RAM.

  • Gradle - Gradle is used to build the jobs and run it locally or remotely. It can be downloaded from https://gradle.org/gradle-download/ (minimum version 2.8)

  • Hadoop environment (Required for running jobs locally):
    Hadoop Build for Windows - 32 Bit
    Hadoop Build for Windows - 64 Bit

Setting up Environment Variables: 

  • Gradle env variable:
    GRADLE_USER_HOME=<Gradle installation directory>
    PATH=%GRADLE_USER_HOME%\bin;
  • HADOOP_HOME environment variable:
    HADOOP_HOME= <hadoop installation directory>
    PATH=% PATH%;%HADOOP_HOME%\bin;
    Make sure that GRADLE and HADOOP bin directories have execute permission.
  • Set JDK_HOME=<jdk 1.7 home directory >
    PATH=% PATH%;% JDK_HOME%\bin;
