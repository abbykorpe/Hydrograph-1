Input Hive Textfile

Hydrograph Release Version 1.0

Input Hive Textfile as the name suggests accepts a Text file as input source and is present under the Input category in the component palette

This component is used to run a Hive Job and accepts Text file format as input. It accepts the hive database name and table name from the user in the component properties and has a single output port.

The Input Hive Textfile component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
