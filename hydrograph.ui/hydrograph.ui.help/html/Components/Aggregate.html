Aggregate

Hydrograph Release Version 1.0

Aggregate component as the name suggests provides aggregate operations on the input and is present under the Transform category in the component palette. Aggregations are performed on groups of data as desired by the user. It contains only one input and one output port.

User is provided with options to create Java class that specify the aggregate operation which needs to be performed. Optionally, user can also use the pre-defined standard Operation Class provided in the Operation Class section of the Help content.

The Aggregate component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
