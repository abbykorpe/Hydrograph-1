Pre-requisites for Macintosh

Setup Requirements on developer machine

  • System requirements
    • Macintosh (64 bit)
    • 8 GB RAM(minimum, 2 GB is required by Hydrograph IDE)
  • Software requirements
    • JDK 1.8, 64bit
    • Gradle 3.0(minimum, used to build the jobs and run it locally or remotely. Download
    • Hadoop, 64bit(required for running jobs locally)

After installing the required software, configure following values in System Environment variables:

  • export JAVA_HOME = < jdk 1.8 home directory path >
  • export HADOOP_HOME = < hadoop installation directory path >
  • export PATH=$PATH:$JAVA_HOME/bin:<gradle installation directory>/bin:$HADOOP_HOME/bin

Make sure that GRADLE and HADOOP bin directories have execute permission to the user running Hydrograph.
