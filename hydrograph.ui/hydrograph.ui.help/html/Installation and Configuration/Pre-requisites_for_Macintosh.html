Pre-requisites for Macintosh

Hydrograph Release Version 1.0

Setup Requirements on developer machine: Operating system – Macintosh 64 bit

  • JDK 1.7 64 bit

  • RAM: Hydrograph tool requires minimum 1 GB of RAM hence developer machine should have minimum 4 GB RAM.

  • Gradle - Gradle is used to build the jobs and run it locally or remotely. It can be downloaded from https://gradle.org/gradle-download/ (minimum version 2.8)

Setting up Environment Variables: 

  • export GRADLE_USER_HOME=<Gradle installation directory>

  • export HADOOP_HOME= <hadoop installation directory>

  • export JAVA_HOME=<Java home directory >

  • export PATH=$PATH:$GRADLE_USER_HOME/bin:$JAVA_HOME/bin:$HADOOP_HOME/bin

Please verify that GRADLE and HADOOP bin directories have execute permission to the user running Hydrograph.
