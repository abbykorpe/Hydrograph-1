Input File Delimited

Hydrograph Release Version 1.0

Input File Delimited as the name suggests accepts a delimited file as input source and is present under the Input category in the component palette

This component is used to read a delimited text file. It accepts the delimiter from the user in the component properties and has a single output port.

The Input File Delimited component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
