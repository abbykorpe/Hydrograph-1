Generate Records

Hydrograph Release Version 1.0

Generate Records as the name suggests accepts number of records along with schema definition for each field and generated Dummy records. These records can be further used for testing purposes.

It accepts the schema from the user in the component schema tab and has a single output port.

The Generate Records component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
