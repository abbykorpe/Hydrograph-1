Run Configuration

Hydrograph Release Version 1.0

The Run Configuration provides the user with options to either run a job on Local or Remote mode.

On selecting the Local mode, the user doesn't need to provide any additional details. The job will be launched from the product and executed on the user's local machine.

On selecting the Remote mode, the user will need to provide details of the Hadoop Cluster Edge Node, it's Host name, username and password. User will additionally have to specify location of the Run utility that is used to run the job on cluster. This run utility will be required to be placed on the cluster before hand.

Path of the run utility, path to place the XML's, Jar files and parameter files need to be specified in the Run configuration grid.

Option to edit the Run configuration is present in the tool bar. Also, the Run configuration grid opens up when user runs the job using the Run button.
